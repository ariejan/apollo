# Decisions Needed

This file tracks decisions that require human input. Claude Code will add entries here when blocked on choices that need human judgment.

---

## How to Use This File

### For Claude Code:
1. Add new decisions at the top (below the template)
2. Include all relevant context and options
3. Mark blocked tasks in TASKS.md
4. Continue working on unblocked tasks

### For Humans:
1. Review pending decisions
2. Add your decision in the "Resolution" section
3. Change status to RESOLVED
4. Claude Code will pick up the decision on next run

---

## Template

```markdown
## [YYYY-MM-DD] Decision: <Title>

**Context:** 
<Why this decision is needed>

**Options:**
1. **Option A** - <description>
   - Pros: ...
   - Cons: ...
2. **Option B** - <description>
   - Pros: ...
   - Cons: ...

**Recommendation:** 
<Claude's suggested approach and reasoning>

**Blocked Tasks:** 
- Task 1
- Task 2

**Status:** PENDING | RESOLVED

**Resolution:** 
<Human's decision and any additional notes - filled in by human>
```
